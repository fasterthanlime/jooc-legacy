Pointers

Pointers are used in ooc exactly as in C, although they are not as welcome. They're used extensively behind the scenes, to implement classes, etc, but their use is discouraged when not necessary. A good use case for using pointers (instead of a objects, for example) is low-level implementation of a speed-critical algorithm. Using pointers is to ooc what assembler is to C: something you shouldn't use unless you really need it for speed.
